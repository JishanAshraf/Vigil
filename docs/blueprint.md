# **App Name**: CommunityHub

## Core Features:

- Report Issue: Users can report local issues by providing a description, images, and location.
- View Local Alerts: Display a feed of alerts posted by users in the neighborhood.
- Authority Tagging: Notify relevant local authorities about an issue based on its category and location.
- Health Symptom Checker: Users can input symptoms and receive information on possible conditions and general advice. AI acts as a tool that cross-references reported symtoms to suggest remedies and over-the-counter medication. DISCLAIMER PROVIDED.
- Authentication: Users can sign up and log in using email address and phone number. SMS OTP verification implemented.

## Style Guidelines:

- Primary color: Blue (#29ABE2) to inspire trust and security within the community.
- Background color: Light blue (#E5F6FD) to provide a clean, calming backdrop.
- Accent color: Green (#90EE90) to signal resolution and positive action.
- Body and headline font: 'PT Sans', a sans-serif with a modern and slightly warm feel.
- Use clear and recognizable icons to represent different categories of issues and actions.
- A card-based layout for the alerts feed, prioritizing key information for quick scanning.
- Subtle animations for new alerts appearing and status updates, providing clear visual feedback.
