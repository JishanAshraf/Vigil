## Installation

Build VIGIL from source and install dependencies:

1. **Clone the repository**  
```bash
git clone https://github.com/JishanAshraf/Vigil
````

2. **Navigate to the project directory**

```bash
cd Vigil
```

3. **Install dependencies using npm**

```bash
npm install
```
